# 资金管理系统

资金管理系统，用于管理各个系统之间的资金审核流程。

### 文档

- [开发规范](http://doc.fe.jyb.com/book/style-guide/index.html)
- [jfet 文档](http://doc.fe.jyb.com/book/workflow/index.html)

### 本地开发

```bash
$ npm i
$ npm run dev
```

### 构建

*** 集成开发环境 ***

```bash
$ npm run build:dev
```

*** 集成测试环境 ***

```bash
$ npm run build:test
```

*** 预发布环境 ***

```bash
$ npm run build:pre
```

*** 生产环境 ***

```bash
$ npm run build:prod
```
